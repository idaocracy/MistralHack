
Aiscreenu is an overlay app specifically designed for games. It gives you a handful of accessibility features directly on the screen while you're playing. The following mockup was made using Unity. Click the screen to progress.  
On the bottom right you can see buttons for each of the feature that was developed during the Mistral hackathon. 

**OCR**
By clicking the OCR button, it will take the current screen view and use OCR to detect and highlight characters. You can click on each highlight to get more information about the word. Click the X on the top right to remove the highlights.

**Voice controller**
By pressing the REC button, the game will start continuously record your voice input, making it possible to control the game using your voice. Press the button and say "Continue" to continue.

**TTS**
And finally we have the TTS button. It will take the dialogue text and turn it into speech. Try it now.

The mockup used Mistral, Pixtral and Voxtral to bring this mockup to fruition. The app may be expanded to include a desktop AI companion, real-time options adjustment and other cool features that are possible with AI.
